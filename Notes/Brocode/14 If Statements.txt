Topic:
    If statements

Keywords:
    if if else

Notes:
    Else if statments allow us to check the conditions of variables (see below).
        let age = 18;
        if (age >= 65) {
            console.log("You are a senior citizen!");
        } else if (age >= 18) {
            console.log("You are an adult!");
        } else if (age < 0) {
            console.log("You have not been born yet!");
        } else {
            console.log("You are not an adult!");
        }

Summary:
    A basic form of decision making if a condition is true, then do something, if not do nothing!
